Logo Usage Guidelines

  • Primary Logo: Use the horizontal version for most applications including business cards, letterheads, and website headers
  • Background Versions: Use circle background versions for app icons, social media profiles, and when you need a contained logo
  • Symbol Only: Use for favicons, watermarks, and when space is extremely limited
  • Vertical Layout: Use for vertical layouts, tall banners, and narrow spaces
  • Dark Background: Always use white/light versions on dark backgrounds
  • Monochrome: Use for single-color printing, newspapers, and cost-effective applications
  • Minimum Size: Logo should never be smaller than 40px in height for digital use, 0.5 inches for print
  • Clear Space: Maintain clear space around the logo equal to half the logo's height
